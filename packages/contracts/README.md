# Shared Contracts

This package stores artifacts shared between backend and frontend agents.

- `fixtures/`: JSON payloads the frontend can use while the backend is still in mock or live-development mode.
- FastAPI OpenAPI output: generated by running `apps/api` and reading `/openapi.json`.

Do not include secrets or real FortiGate host values in this package.

Auth fixtures model the response bodies only. Cookie details are owned by FastAPI and covered by backend tests.
